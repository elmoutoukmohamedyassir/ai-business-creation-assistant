# AI Business Creation Assistant

Frontend MVP for a PFE project focused on helping entrepreneurs navigate business creation and receive post-launch support through an AI-assisted platform.

## Overview

This project explores how AI and LLM-powered assistance can support entrepreneurs during:

- Business creation planning  
- Administrative and operational roadmap tracking  
- Guided task execution  
- Post-launch business monitoring  
- Recommendation and decision support  

Current repository contains the **Frontend MVP (in progress)**.

## Current Stage (MVP Frontend)

Implemented / being built:

- Landing page
- Authentication UI (frontend only)
- Business project intake form
- Entrepreneur dashboard
- Creation roadmap interface
- AI assistant chat interface (mock)
- Post-creation updates page
- Recommendations page

## Project Vision

Build an intelligent assistant that helps entrepreneurs:

1. Structure company creation steps  
2. Track progress and priorities  
3. Ask contextual questions through an LLM assistant  
4. Monitor early business indicators  
5. Receive actionable recommendations
